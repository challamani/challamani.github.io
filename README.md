## Local Development

To run this site locally:

1. Install Ruby and Bundler
2. Clone the repository
3. Install dependencies:
   ```bash
   bundle install
   ```
4. Run the local server:
   ```bash
   bundle exec jekyll serve
   ```
5. Visit `http://localhost:4000` in your browser
